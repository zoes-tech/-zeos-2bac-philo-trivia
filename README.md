# Zeos 2Bac Trivia

Interactive philosophy trivia for Moroccan 2nd Baccalaureate students.

## Features

- 349 Questions across 3 levels
- Answer Shuffling (Fisher-Yates)
- Timer System (+3s bonus)
- Streak Multipliers (up to 2.0x)
- God Mode at 10+ streak
- Local Storage (no cloud)
- PWA - Installable

## Quick Deploy

### Vercel (Free)

```bash
# 1. Push to GitHub
git push -u origin main

# 2. Go to https://vercel.com
# 3. Import and Deploy
```
