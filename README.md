## 🤔 What is Zeos?

Zeos 2Bac Trivia is an open-source, gamified learning platform designed to help Moroccan 2nd Baccalaureate students master philosophy through fun, interactive trivia - without the friction of logins or accounts.

**Built for students, by students** 🇲🇦

## ✨ Features

- **349 Questions** across 3 educational levels
- **Answer Shuffling** - Fisher-Yates algorithm prevents pattern memorization
- **Timer System** - 5-10 seconds per question (+3s bonus for correct answers)
- **Streak Multipliers** - Up to 2.0x for consecutive correct answers
- **God Mode** - Activates at 10+ streak with special effects
- **Ranking System** - Moroccan-themed ranks
- **Local Storage** - All progress saved locally (no cloud, full privacy)
- **PWA** - Installable, works offline
- **RTL Arabic** - Full Arabic language support
